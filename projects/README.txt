IDE's projects for easily starting development
----------------------------------------------
The folder content:

 - boos.exe.appl.core.target.atmel.sama5d3x.elf.eclipse.gcc
    The Eclipse IDE project for starting development new application
    Atmel SAMA5d3x Microprocessors family.

 - boos.exe.appl.core.target.demo.elf.eclipse.gcc
    The Eclipse IDE project for starting development new target kernel
   
 - boos.exe.appl.core.target.ti.am18x.coff.ccstudio
    The Code Composer 3.3 IDE project for starting development new application
    Texas Instruments AM18x ARM Microprocessors family
   
 - boos.exe.appl.core.target.ti.am18x.elf.eclipse.gcc
    The Eclipse IDE project for debugging pedantic mode
    Texas Instruments AM18x ARM Microprocessors family 
 
 - boos.exe.appl.core.target.ti.tms320c64x.coff.ccstudio
    The Code Composer 3.3 IDE project for starting development new application
    Texas Instruments TMS320C64x Digital Signal Processors family
 
 - boos.exe.appl.core.target.ti.tms320c64x.elf.eclipse.gcc
    The Eclipse IDE project for debugging pedantic mode
    Texas Instruments TMS320C64x Digital Signal Processors family

 - boos.exe.appl.core.target.ti.tms320f2833x.coff.ccstudio
    The Code Composer 3.3 IDE project for starting development new application
    Texas Instruments TMS320F2833x Digital Signal Controller family
    
 - boos.exe.appl.ti.tms320c64x.coff.ccstudio
    The Code Composer 3.3 IDE project for starting development new application by using built libraries
    Texas Instruments TMS320C64x Digital Signal Processors family    

 - boos.lib.core.ti.am18x.coff.ccstudio
